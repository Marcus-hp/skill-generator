# Skill Generator — 交互式技能包生成向导（增强版）

通过对话问答，引导你一步步创建完整的 Agent Skill 文件包。

**🚀 新增功能：**
- 💾 **会话进度保存** - 支持中断后恢复
- 🛡️ **智能错误处理** - 友好的错误提示和恢复建议  
- 🧠 **智能推荐系统** - 自动推荐模板、名称和关键词
- 👁️ **实时预览编辑** - 边创建边预览，支持实时修改
- 🔧 **技能更新管理** - 支持修改已有技能，版本控制
- ✅ **严格验证机制** - JSON Schema 验证，确保质量

## 安装

将此文件夹放到项目的 `.claude/skills/` 目录下：

```
你的项目/
└── .claude/
    └── skills/
        └── skill-generator/   ← 放这里
```

## 使用方式

在 Claude Code 中说：

> "帮我创建一个新的 Skill"  
> "我想做一个技能包"  
> "新建 skill"
> "编辑已有技能"  
> "恢复之前的会话"

向导启动后，通过 **智能问答** 完成创建：

| 步骤 | 内容 | 增强功能 |
|------|------|----------|
| 1 | 技能定位 | 智能类型识别 + 恢复会话选项 |
| 2 | 技能命名 | AI推荐名称 + 置信度评分 |
| 3 | 触发描述 | 智能起草 + 关键词优化 |
| 4 | 依赖工具 | 兼容性检查 + 自动推荐 |
| 5 | 输入/输出 | 逻辑验证 + 格式建议 |
| 6 | 执行步骤 | 实时预览 + 逻辑检查 |
| 7 | 注意事项 | 智能建议 + 风险评估 |
| 8 | 附加资源 | 结构验证 + 自动生成 |

**新增交互功能：**
- 🎯 **智能识别** - 自动理解用户意图
- 💬 **自然语言编辑** - "修改描述为..."  
- 📊 **实时预览** - 随时查看技能信息
- 🔙 **会话恢复** - 中断后继续创建
- ⚡ **快捷操作** - 帮助、保存、预览、修改

完成后，Claude 直接帮你把文件创建好。

## 关于 description 字段

`description` 是整个 Skill 最重要的字段，决定了 Claude 什么时候会使用它。
向导会重点引导你完善这个部分，也会帮你起草初稿。

**🧠 智能优化：**
- 自动检测关键词覆盖率
- 推荐中英文触发词
- 优化场景描述的准确性
- 验证描述长度和质量
