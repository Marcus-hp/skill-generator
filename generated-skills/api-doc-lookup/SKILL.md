# api-doc-lookup

快速查询 API/库的用法，直接返回可运行的示例代码。

## 输入

用户提供库名、函数名、或描述想要实现的功能。例如：
- "prisma 的 upsert 怎么用"
- "FastAPI 怎么做文件上传"
- "React Query 的 useInfiniteQuery"

## 输出

结构化的用法说明，包含可直接复制使用的代码示例。

## 执行步骤

1. 识别用户要查的目标：
   - 具体函数/方法 → 给出签名 + 参数说明 + 示例
   - 某个功能如何实现 → 给出最佳实践方案 + 示例
   - 两个方案的对比 → 给出对比表格
2. 检查项目的依赖版本（package.json / pyproject.toml），确保给出的用法与项目版本匹配
3. 输出格式：
   ```
   ## [库名] - [函数/功能名]

   **版本：** x.y.z（你的项目使用的版本）

   ### 基本用法
   [最简代码示例]

   ### 常见用法
   [覆盖常用场景的代码示例]

   ### 参数说明
   | 参数 | 类型 | 必填 | 说明 |
   |------|------|------|------|

   ### 注意事项
   - [坑点和最佳实践]
   ```
4. 如果用户需要，直接将示例代码插入到当前文件的合适位置

## 注意事项

- 优先参考项目中已有的使用方式，保持风格一致
- 版本很重要：不同大版本 API 可能完全不同，一定要匹配项目版本
- 如果不确定某个 API 的最新行为，明确告知用户核实官方文档
- 常用库速查范围包括但不限于：
  - TS: React, Next.js, Prisma, Zod, tRPC, TanStack Query, Tailwind
  - Python: FastAPI, SQLAlchemy, Pydantic, pandas, pytest, httpx
