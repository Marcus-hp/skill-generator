# git-commit-standard

根据 git diff 自动生成符合 Conventional Commits 规范的提交信息。

## 安装

**Cursor：** 将此文件夹放到 `~/.cursor/skills-cursor/` 或项目的 `.cursor/skills/` 下。

**Claude Code：** 将此文件夹放到项目的 `.claude/skills/` 下。

## 使用方式

完成代码修改后，直接说：

> "帮我提交"
> "commit"
> "提交一下"

AI 会自动分析 diff 并生成规范的 commit message。

## 依赖

无特殊依赖，需要项目已初始化 git。
