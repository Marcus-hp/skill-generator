# i18n-helper

国际化辅助工具：扫描硬编码字符串、生成翻译文件、管理多语言内容。

## 安装

**Cursor：** 将此文件夹放到 `~/.cursor/skills-cursor/` 或项目的 `.cursor/skills/` 下。

**Claude Code：** 将此文件夹放到项目的 `.claude/skills/` 下。

## 使用方式

> "扫描 src/pages 里的硬编码字符串"
> "这个页面要支持英文"
> "补充日文翻译"

## 依赖

项目需要已安装对应的 i18n 框架（react-i18next / next-intl / gettext 等）。
