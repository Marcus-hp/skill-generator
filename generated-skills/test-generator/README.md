# test-generator

自动生成单元测试，支持 TypeScript（Jest/Vitest）和 Python（pytest）。

## 安装

**Cursor：** 将此文件夹放到 `~/.cursor/skills-cursor/` 或项目的 `.cursor/skills/` 下。

**Claude Code：** 将此文件夹放到项目的 `.claude/skills/` 下。

## 使用方式

> "帮我写 src/utils/auth.ts 的测试"
> "给这个函数补个单元测试"
> "生成 pytest 测试用例"

## 依赖

项目需要已安装对应的测试框架（Jest/Vitest 或 pytest）。
