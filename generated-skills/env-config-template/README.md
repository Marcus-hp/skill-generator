# env-config-template

一键生成 Docker、CI/CD、Linter 等开发环境配置。支持 TypeScript 和 Python 项目。

## 安装

**Cursor：** 将此文件夹放到 `~/.cursor/skills-cursor/` 或项目的 `.cursor/skills/` 下。

**Claude Code：** 将此文件夹放到项目的 `.cursor/skills/` 下。

## 使用方式

> "帮我配个 Dockerfile"
> "搭个 GitHub Actions CI"
> "配一下 ESLint + Prettier"

## 依赖

无特殊依赖。
