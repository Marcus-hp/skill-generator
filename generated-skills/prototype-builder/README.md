# prototype-builder

快速生成可交互的 HTML 原型页面，浏览器直接打开即可预览。也支持生成 UI 设计图片。

## 安装

**Cursor：** 将此文件夹放到 `~/.cursor/skills-cursor/` 或项目的 `.cursor/skills/` 下。

**Claude Code：** 将此文件夹放到项目的 `.claude/skills/` 下。

## 使用方式

> "画一个登录页原型"
> "做个后台 Dashboard"
> "这个页面大概长什么样"
> "出个电商首页的 demo"

## 输出

- HTML 可交互原型：单文件，浏览器打开即可预览
- UI 设计图片：PNG 格式

## 依赖

无特殊依赖。HTML 原型使用 Tailwind CSS CDN，需要联网。
