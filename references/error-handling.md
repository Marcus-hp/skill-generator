# 错误处理规范

## 基本错误处理

- **用户输入不在选项内** → 友好提示，重新展示选项，提供输入建议
- **用户说"回到上一步"** → 支持，重新展示上一步
- **用户说"重来"** → 从第 0 步开始
- **用户跳步骤** → 支持，跳过的步骤用合理默认值填充并告知用户
- **会话中断** → 自动保存进度，下次启动时可选择恢复
- **输入验证失败** → 提供具体的错误信息和修复建议
- **文件操作错误** → 检查权限、路径有效性，提供备选方案

## 增强的用户交互

**支持的快捷命令：**
- "帮助" / "help" → 显示当前步骤的帮助信息
- "保存" / "save" → 手动保存当前进度
- "预览" / "preview" → 查看当前技能信息摘要
- "修改" / "modify" → 修改之前步骤的内容
- "跳过" / "skip" → 跳过当前可选步骤
- "退出" / "quit" → 保存进度并退出

**智能输入识别：**
- 自动识别用户意图（返回、重做、帮助等）
- 容错处理，支持模糊表达
- 提供输入建议和示例
