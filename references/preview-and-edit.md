# 技能预览和实时编辑

## 实时预览功能

在任何步骤，用户都可以说"预览"查看当前技能的完整信息：

```
📋 **技能名称** - 完成度 75%
[████████████████░░] 75%

📝 **简介**: [技能简介]

🎯 **触发描述**:
```
[完整的触发描述]
```

🔄 **输入输出**:
- **输入**: [输入描述]
- **输出**: [输出描述]

📦 **依赖库**:
- [依赖库列表]

⚡ **执行步骤**:
1. [步骤1]
2. [步骤2]
...

📁 **文件结构**:
📄 SKILL.md - 核心技能描述
📄 README.md - 使用说明文档
[其他文件结构]

🔍 **待完善**:
- [缺失字段列表]
```

## 实时编辑功能

用户可以通过自然语言进行编辑：

**编辑类型：**
- "修改描述：[新描述]" → 更新触发描述
- "添加步骤：[新步骤]" → 添加执行步骤
- "删除步骤：第N步" → 删除指定步骤
- "添加依赖：[库名]" → 添加依赖库
- "修改输入：[新输入描述]" → 更新输入描述

**批量编辑：**
- "修改名称为：[新名称]"
- "设置简介为：[新简介]"
- "清空所有步骤，重新开始"

## 智能编辑建议

系统会根据当前完成度提供编辑建议：

```
💡 **编辑建议**：
1. 建议添加技能简介，让用户快速了解功能
2. 考虑是否需要添加依赖库
3. 执行步骤可以更详细一些
4. 在描述中明确说明触发关键词
```

## 验证和错误提示

实时验证技能数据，提供具体错误信息：

```
⚠️ **数据验证发现问题**：
- 触发描述太短，建议至少50个字符
- 技能名称只能包含小写字母、数字和连字符
- 至少需要一个执行步骤

💡 **修复建议**：
1. 扩展触发描述，包含更多场景和关键词
2. 修改技能名称格式
3. 添加详细的执行步骤
```
