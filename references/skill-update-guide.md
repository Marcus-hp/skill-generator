# 技能更新管理

## 编辑已有技能流程

当用户选择"编辑已有技能"时：

1. 调用 `scripts/skill_updater.py` 列出所有已有技能
2. 展示技能列表供用户选择：
```
🔧 找到以下已有技能：

1️⃣  pdf-creator — PDF文档生成工具 — 2天前
2️⃣  data-analyst — 数据分析助手 — 1周前  
3️⃣  code-helper — 代码审查工具 — 1个月前

请回复数字选择要编辑的技能，或说"返回"：
```

3. 加载选中技能的数据，显示当前信息
4. 提供编辑选项：
```
📝 **pdf-creator** 当前信息：

🎯 描述：[当前描述]
📦 依赖：[当前依赖]
⚡ 步骤：[当前步骤]

编辑选项：
1️⃣  修改基本信息（名称、描述）
2️⃣  更新依赖库
3️⃣  调整执行步骤
4️⃣  添加注意事项
5️⃣  重命名技能
6️⃣  查看变更历史
7️⃣  删除技能

请选择操作：
```

## 版本管理

**自动备份：**
- 每次修改前自动创建备份
- 保留最近10个版本的备份
- 支持手动创建备份点

**变更历史：**
```
📜 **pdf-creator** 变更历史：

v3 (2024-03-01 14:30) - 添加PDF加密支持
  + 修改描述
  + 添加依赖：cryptography
  + 更新步骤

v2 (2024-02-28 10:15) - 优化性能
  + 修改步骤3和4
  + 更新注意事项

v1 (2024-02-25 16:45) - 初始版本
  + 创建技能
```

**版本比较：**
- 支持比较任意两个版本的差异
- 显示字段级别的变更
- 提供合并建议

**版本恢复：**
- 可恢复到任意历史版本
- 恢复前自动备份当前版本
- 支持预览恢复效果

## 高级编辑功能

**批量操作：**
- 批量更新多个技能的相同字段
- 批量添加依赖库
- 批量修改描述模板

**模板应用：**
- 将新模板应用到现有技能
- 保留自定义内容
- 智能合并冲突

**导入导出：**
- 导出技能配置为JSON
- 从配置文件创建技能
- 技能包分享功能

## 安全机制

**操作确认：**
- 删除操作需要二次确认
- 重大修改显示影响预览
- 支持撤销最近操作

**数据验证：**
- 修改前验证数据格式
- 检查依赖库兼容性
- 验证技能名称唯一性

**权限控制：**
- 只能修改自己创建的技能
- 系统技能需要特殊权限
- 操作日志记录
